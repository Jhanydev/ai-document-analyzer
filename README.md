# AI Document Analyzer

This project demonstrates a simple automation using Python and AI.

The system reads a PDF document and generates a summary using an AI model.

## Technologies

- Python
- OpenAI API
- PyPDF2
- Automation

## Project Goal

Demonstrate how AI can automate document analysis and reduce manual reading time.
